## 什么是搬瓦工 CN2？为什么测速的结果差异这么大

**CN2**（China Telecom Next Carrier Network）是中国电信的精品骨干网，比普通 163 线路高一个档次，抗拥堵能力更强，晚高峰掉速更少。搬瓦工（BandwagonHost）是加拿大 IT7 Networks 旗下的 VPS 商家，目前在售的中国优化线路主要分三档：

- **CN2 GT**：CN2 线路，但回程部分节点会切到普通 163 骨干，高峰期偶有抖动，价格最便宜
- **CN2 GIA**：全程走中国电信最高等级专线，去程回程都不走普通网络，三网（电信/联通/移动）晚高峰依然稳
- **香港 CN2 GIA**：物理距离近，延迟直接压到 9ms 左右（广东方向），是目前对国内延迟最低的选择，价格也最贵

说实话，**测速结果差异大，主要原因不是搬瓦工"飘忽不定"，而是你在什么时间测、用什么运营商测、测的是哪个机房**。同一台机器，白天空闲时和晚上 8 点高峰期跑出来的数据可以差 30%，这是正常的，不是商家坑你。

**怎么用这些 IP？**

Windows 下打开命令提示符，直接输：


ping 167.88.61.1


把 IP 换成你想测的机房地址，看 `time=` 后面的数字，那就是延迟，单位毫秒（ms）。

丢包怎么看？跑完 ping 之后看最后一行 `Packets: Sent = X, Received = Y, Lost = Z`。如果 Lost 超过 5%，这个机房方向对你的网络不太友好。

CN2 GIA 线路的天然受益者。DC6（USCA_6）测出来的延迟大概在 150-170ms 之间，晚高峰下载跑 100Mbps+ 是正常水平。DC9 类似，两个机房差别不大，选哪个都行。

### 联通用户

### 移动用户

DC6/DC9 现在接了移动 CMIN2（AS58807）精品骨干，实测延迟大概 150ms，晚高峰丢包率控制得不错。这是移动用户在搬瓦工上历史最好的线路配置了。

## 测速有两种方式：买前测延迟 vs 买后测带宽

这是两件不同的事，很多人搞混了。

### 买前：用测试 IP 测延迟

就是上面说的 ping，只需要目标机房的一个 IP，不用买 VPS。这一步的目的是确认这个机房的延迟在你能接受的范围内，以及有没有明显丢包。

**一个小技巧**：单点 ping 不够准确，建议用 ping.chinaz.com（站长工具）输入测试 IP，可以看到全国各地的延迟分布，电信/联通/移动三个运营商的情况一目了然，比你本地 ping 一次可靠得多。

### 买后：用演示站或 SpeedTest CLI 测带宽

搬瓦工给 DC6 和 DC9 机房分别搭了演示站，直接访问就能在浏览器跑 SpeedTest，测出下载/上传速度、延迟波动。这才是真正的带宽测试，延迟 160ms 但带宽跑满，实际用起来比延迟 120ms 但带宽 10Mbps 要舒服得多。

已经买了 VPS 的话，SSH 登录进去跑下面这个命令，测到国内节点的实际带宽：

bash
curl -s https://packagecloud.io/install/repositories/ookla/speedtest-cli/script.deb.sh | sudo bash
sudo apt-get install speedtest
speedtest


还可以用 `backtrace` 脚本看回程路由，确认你的 VPS 是否真的走了 CN2 线路——路由里出现 `59.43.x.x` 的节点，那就是 CN2 GIA 没错。

## 怎么在 KiwiVM 后台切换机房

这是搬瓦工一个很好用的功能，CN2 GIA-E 套餐支持在 14 个机房之间切换，不用重新买。

1. 登录 KiwiVM 控制面板
2. 找到左侧 **Migrate to another DC**（迁移机房）
3. 从列表里选目标机房，确认迁移

迁移会短暂中断服务，新机房 IP 会变。迁完用 `backtrace` 脚本重新确认一下回程线路，确保走的是 CN2 GIA。

注意：香港 CN2 GIA 套餐是独立套餐，无法通过 CN2 GIA-E 套餐迁移过去，两者是不同系列。

## FAQ：关于搬瓦工 CN2 测速的常见问题

**Q：测试 IP ping 延迟 200ms，是不是太高了？**

A：中美物理距离决定了洛杉矶机房延迟大概在 150-200ms，这个范围是正常的。晚高峰到 200ms 也属于可接受区间。如果经常超过 250ms 或者丢包超过 5%，才需要考虑换机房。

**Q：CN2 GIA 和 CN2 GIA-E 有什么区别？**

A：GIA-E 是 GIA 的扩展版（ECOMMERCE 版本），带宽更大，可切换机房更多，实际体验上两者都走 CN2 GIA 精品网，差别主要在带宽上限和套餐灵活性。大多数用户选 GIA-E 更划算。

**Q：移动用户用搬瓦工 CN2 好吗？**

A：2025 年在 DC6/DC9 接入 CMIN2 线路之后，移动用户的体验明显改善了，现在选 CN2 GIA-E 套餐、指定 DC6 或 DC9 机房就行。以前移动用户买搬瓦工确实体验差一截，现在好多了。

**Q：买之前一定要先测速吗？**

A：建议测一下，五分钟的事。尤其是如果你是移动宽带，或者在部分延迟较高的省份，先 ping 一下比买完了后悔要省事。搬瓦工官方提供了测试节点，不用买账号。

**Q：CN2 GIA-E 套餐可以切换到香港机房吗？**

A：不可以。香港 CN2 GIA 是独立套餐系列，无法通过 CN2 GIA-E 套餐迁过去。想用香港机房只能单独购买香港套餐。
